# SkyFitness

SkyFitness is an Android mobile application that connects athletes and trainers on a single platform. The application is developed using **Java** and utilizes **Firebase** for authentication, data storage, and real-time operations. With SkyFitness, athletes can track their workouts, record body measurements, calculate nutritional needs, and communicate with trainers, while trainers can monitor athletesâ€™ progress and manage training programs.

## Features

- **User Authentication:** Secure user registration and login using Firebase Authentication (email & password).
- **Workout Programs:** View and follow workout programs designed for different fitness levels.
- **Body Measurement Tracking:** Record and monitor body measurements such as weight, waist, arm, and leg size.
- **Nutrition Calculator:** Calculate daily protein and carbohydrate requirements.
- **Messaging System:** Real-time chat functionality between athletes and trainers.
- **Firebase Integration:** Cloud Firestore is used for storing workouts, measurements, nutrition data, and messages.
